# 🎬 VEO Automation v3.1.6 - Google Flow AI VEO 自动化 [![English](https://img.shields.io/badge/English-blue)](README.md) [![Tiếng Việt](https://img.shields.io/badge/Tiếng%20Việt-green)](README_vi.md)

**VEO Automation** 是一个强大的 Chrome 扩展程序，可自动化 Google Flow AI VEO3 上的批量视频和图像生成。同时处理多个提示词，配置您的工作流程，并轻松自动下载生成的内容。

-----

## ✨ 主要功能

* **🚀 队列支持：** 将多个提示词添加到等待列表，而不是手动逐个操作。工具将自动逐个处理任务。
* **📝 文本转视频：** 支持导入包含数百个提示词的 `.txt`、`.xlsx` 或 `.csv` 文件，从文本批量创建视频。
* **🖼️ 帧转视频：** 使用静态图片并添加自动运动效果。支持为每个提示词仅使用开始帧或同时使用开始帧和结束帧。
* **🎨 组件化视频：** 将 UI 组件、界面元素或角色图片动画化为视频。支持根据文件名自动匹配角色图片。
* **🖼️ 文本转图片：** 从文本描述生成图片。
* **🔄 图片转图片：** 使用 AI 和文本提示词转换和增强图片。
* **🤖 智能体自动化：** 使用高级智能体控制与 Google Flow AI 智能体合作自动执行任务。
* **📂 从电子表格导入：** 直接从电子表格文件（`.xlsx` 或 `.csv`）导入提示词列表，支持工作表与列的预览选择。
* **💾 自动下载：** 渲染完成后立即自动将视频（720p、1080p、4K）和图片（1k、2k、4k）保存到您的计算机。支持按项目名称组织文件夹。
* **🛡️ 重试机制：** 遇到网络错误或 Google 创建错误时自动重试（可自定义多达 20 次重试）。
* **🌐 多语言界面：** 支持 20 种语言，包括英文、中文、越南语等。

## 📖 使用指南

### 开始使用

1. **导航到 Google Flow**
   - 打开 [Google Labs Flow](https://labs.google/fx/tools/flow)
   - 扩展程序仅在 Flow 项目页面上工作。

2. **打开扩展程序**
   - 点击 Chrome 工具栏中的扩展程序图标并固定，以便后续快速使用。

3. **选择模式**
   - 在控制标签页中从 6 种生成模式中选择。

---

### 1. 文本转视频模式

1. 在控制标签页中选择 **文本转视频** 模式。
2. 在输入框中输入提示词（每个提示词用空行分隔）或点击 **上传 .txt** 或 **上传 .xlsx / .csv** 按钮上传提示词列表。
3. 配置时长设置（可选择 4s、6s、8s、10s 或合并连接模式）。
4. 点击 **运行** 开始批量处理。

**提示词示例：**
```
平静海洋上宁静的日落，轻柔的波浪。
相机慢慢扫过地平线。

夜晚繁华的城市街道，霓虹灯闪烁。
汽车和行人在场景中穿行。
```

---

### 2. 帧转视频模式（原图片转视频）

1. 选择 **帧转视频** 模式。
2. 点击上传区域上传图片（支持一次选择多张图片）。
3. 为每张图片输入提示词（用空行分隔）。
4. 在设置中，配置每个提示词是仅使用开始帧还是同时使用开始帧和结束帧。
5. 点击 **运行** 进行处理。

---

### 3. 组件化视频模式（原组件转视频）

1. 选择 **组件化视频** 模式。
2. 上传组件或角色图片。
3. 输入描述动画的提示词。
4. **自动添加上传的角色图片**（可选）：启用后，系统将根据文件名自动匹配角色图片并添加到提示词中。
5. **自动添加角色（Google Flow 功能 ⭐）**：当提示词中提及项目角色名称时，自动选择对应角色（点击 **扫描角色** 按钮同步角色）。
6. **按说话者自动添加语音**：当提示词中提到说话者名称时，自动选择对应的语音。
7. 点击 **运行** 生成。

---

### 4. 文本转图片模式

1. 选择 **文本转图片** 模式。
2. 输入详细的图片描述（每行一个或空行分隔）。
3. 在设置中配置宽高比和图片模型。
4. 点击 **运行** 生成图片。

---

### 5. 图片转图片模式

1. 选择 **图片转图片** 模式。
2. 上传源图片。
3. 输入转换提示词。
4. **自动添加上传的角色图片**（可选）：基于文件名将角色图片自动匹配并引入到提示词。
5. 点击 **运行** 开始转换。

---

### 6. 智能体自动化模式

1. 选择 **智能体自动化** 模式。
2. 输入描述智能体任务的提示词。
3. 支持角色控制（Google Flow 功能 ⭐）和基于提示词提及的按说话人自动添加语音。
4. 点击 **运行** 执行智能体任务。

---

### 📂 电子表格和文件导入

- 在任意模式下，点击 **上传 .xlsx / .csv** 或 **上传 .txt** 批量导入提示词。
- 对于电子表格，会弹出预览窗口，允许您选择目标 **工作表 (Sheet)** 以及包含提示词的 **列**。
- 点击 **导入提示词** 填入到队列中。

---

### 队列管理

* 在 **Prompt 队列** 部分查看待处理任务列表。
* 点击 **移除** 丢弃多余任务，或点击 **停止** 停止当前运行的整个队列。
* 实时显示提示词之间的等待时间倒计时。

## ⚙️ 设置配置

访问 **设置** 标签页进行自定义：

### 常规设置
- **默认模式**：设置您常用的生成模式，以便在打开扩展程序时自动为您选择。
- **默认宽高比**：选择 16:9、9:16、1:1、3:4 或 4:3。
- **每个 prompt 的输出数量**：设置每个提示词要创建的数量（1-4 个图像/视频）。
- **并发 Prompt**：同时处理 1-6 个提示词（注意：合并连接和智能体模式限制 1 个并发，以防任务冲突）。
- **随机等待**：设置提示词提交之间的等待时间，以避免请求速率限制。

### 模型选择
- **模型 (视频)**：从 Veo 3.1 或 Veo 2 变体中选择。
- **图像模型**：选择用于文本转图像生成的 AI 模型。

### 模式特定选项
- **默认视频选项**：可选择 4秒、6秒、8秒、10秒或合并模式（`4秒/6秒/8秒/10秒（合并）`）。
  > [!NOTE]
  > 使用合并连接时，当前的提示词与后续的提示词将被合成为单个视频。序列中的最后一个提示词始终默认使用标准持续时间（非合并）。
- **默认图片模式选项**：设定为 新图片 或 上一张图片（复用前一个提示词生成的图片作为输入）。
- **最大输入图片/视频**：为帧转视频、组件化视频和图片转图片模式配置输入资产的最大限制。

### 下载和高级设置
- **自动下载质量**：选择自动下载的视频分辨率（720p, 1080p, 4K）和图片分辨率（1k, 2k, 4k）。部分质量等级需要 Ultra/Pro 方案。
- **失败时最大重试次数**：设置生成失败时的重试次数（1-20次）。
- **语言**：在 20 种支持的语言中选择。

## 💡 提示和最佳实践

1. **避免请求限制（Rate Limit）**
   - 高峰期建议设置约 2-3 个并发提示词。
   - 适当将随机等待延迟调整为 30 秒或更长。

2. **图片命名规则**
   - 自动匹配时，图片命名保持逻辑性（例如：`hero_pose.png`），并在提示词中提及完全一致的文件名。

3. **修复“异常活动”错误**
   - 在开启自动化运行前，请确保您能够手动在 Google Flow 上连续成功提交至少 3 个提示词且不出现任何错误。

## 🔧 故障排除

| 问题 | 原因和解决方案 |
| --- | --- |
| **扩展程序不工作** | 确保您在 Google Flow 项目页面上 (`labs.google/fx/tools/flow/project/...`)。 |
| **异常活动 / 验证错误** | 1. 点击控制标签页中的 **修复错误**（清除缓存）按钮。<br>2. 尝试新建一个 Flow 项目。<br>3. 尝试使用全新的浏览器配置文件。<br>4. 通过手动创建测试账号是否已被 Google 标记。 |
| **视频/图片不下载** | 关闭 Chrome 的下载弹窗询问：Chrome 设置 -> 下载 -> **关闭** *"下载前询问每个文件的保存位置"*。 |
| **“政策错误”** | 提示词或图片违反了 Google 内容政策。工具将自动跳过并处理下一个提示词。 |
| **屏幕缩小** | 这是工具定位 UI 按钮的自动化缩放功能。请勿在运行时手动调整浏览器缩放级别。 |
