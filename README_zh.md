# 🎬 VEO Automation v2.1.2 - Google Flow AI VEO 自动化 [![English](https://img.shields.io/badge/English-blue)](README.md) [![Tiếng Việt](https://img.shields.io/badge/Tiếng%20Việt-green)](README_vi.md)

**VEO Automation** 是一个强大的 Chrome 扩展程序，可自动化 Google Flow AI VEO3 上的批量视频和图像生成。同时处理多个提示词，配置您的工作流程，并轻松自动下载生成的内容。

-----

## ✨ 主要功能

* **🚀 智能队列：** 将一批提示词或图片添加到等待列表。工具将自动逐个处理任务。
* **📝 文本转视频：** 支持导入包含数百个提示词的 `.txt` 文件。
* **🖼️ 图片转视频：** 自动上传图片、裁剪宽高比并生成运动效果。
* **🎨 组件转视频：** 将 UI 组件和界面元素动画化为视频。
* **🖼️ 文本转图片：** 从文本描述生成图片。
* **🔄 图片转图片：** 使用 AI 和文本提示词转换和增强图片。
* **💾 自动下载：** 渲染完成后立即自动将视频和图片保存到您的计算机。支持按项目名称组织文件夹。
* **🛡️ 重试机制：** 遇到网络错误或 Google 队列已满时自动重试。
* **⚙️ 深度自定义：**
    * 选择模型（Veo 2、Veo 3.1...）。
    * 选择宽高比（16:9、9:16、1:1）。
    * 设置每个提示词的视频变体数量（1-4 个变体）。
    * 配置并发提示词（1-6 个同时）。
    * 设置提示词延迟（0-300 秒）。

## 📖 使用指南

### 开始使用

1. **导航到 Google Flow**
   - 打开 [Google Labs Flow](https://labs.google/fx/tools/flow)
   - 扩展程序仅在 Flow 项目页面上工作

2. **打开扩展程序**
   - 点击 Chrome 工具栏中的扩展程序图标
   - 或使用侧边栏（如果已启用）

3. **选择模式**
   - 在控制标签页中从 5 种生成模式中选择
   - 每种模式都有特定的功能和需求

### 1. 文本转视频模式

1. 在控制标签页中选择**文本转视频**模式。
2. 在空框中输入提示词（每个提示词用空行分隔）或点击**导入文件 (.txt)** 按钮上传提示词列表。
3. 设置视频数量和宽高比。
4. 点击**运行**开始批量处理。
5. 在进度部分监控进度。

**提示词示例：**
```
平静海洋上宁静的日落，轻柔的波浪。
相机慢慢扫过地平线。

夜晚繁华的城市街道，霓虹灯闪烁。
汽车和行人在场景中穿行。
```

### 2. 图片转视频模式

1. 选择**图片转视频**模式。
2. 点击**选择图片**上传（支持一次选择多张图片）。
3. 为每张图片输入提示词（用空行分隔）。
4. 配置时长设置（8 秒或合并）。
5. 点击**运行**进行处理。

**提示：**
- 每张图片都会与每个提示词一起处理
- 按名称、日期或自定义顺序排序图片
- 每个提示词最多 2 张图片（可在设置中配置）

### 3. 组件转视频模式

1. 选择**组件转视频**模式。
2. 上传组件图片。
3. 启用"自动添加角色图片"（可选）- 根据文件名自动匹配角色图片。
4. 输入描述动画的提示词。
5. 点击**运行**生成视频。

**功能：**
- 根据文件名自动匹配角色图片
- 每个提示词最多支持 3 张图片
- 非常适合为 UI 组件制作动画

### 4. 文本转图片模式

1. 选择**文本转图片**模式。
2. 输入详细的图片描述。
3. 在设置中配置宽高比和图片模型。
4. 点击**运行**生成图片。

### 5. 图片转图片模式

1. 选择**图片转图片**模式。
2. 上传源图片。
3. 输入转换提示词。
4. 配置设置（每个提示词最多 10 张图片）。
5. 点击**运行**转换图片。

### 队列管理

* 在提示词队列部分查看待处理任务列表。
* 您可以**移除**多余任务或**停止**正在运行的任务。
* 监控每个提示词的实时进度。

## ⚙️ 设置配置

访问**设置**标签页进行自定义：

### 常规设置
- **默认模式**：设置您首选的生成模式
- **默认宽高比**：选择 16:9、9:16 或 1:1
- **每个提示词的输出**：1-4 个输出
- **并发提示词**：1-6 个同时提示词
- **提示词延迟**：提示词之间的等待时间（0-300 秒）

### 模型选择
- **视频模型**：从 Veo 3.1 或 Veo 2 变体中选择
- **图片模型**：选择用于文本转图片的 AI 模型

### 下载设置
- **自动下载质量（视频）**：720p、1080p 或不下载
- **自动下载质量（图片）**：1k、2k、4k 或不下载
- 视频保存到 Chrome 的下载文件夹
- 每个项目都有自己的文件夹

### 高级设置
- **最大重试次数**：失败时 1-20 次重试尝试
- **默认视频帧**：8 秒或合并
- **每个提示词的最大图片数**：为每种模式配置
- **语言**：英语、越南语或中文

## 💡 提示和最佳实践

1. **提示词编写**
   - 具体说明风格、时长和视觉元素
   - 使用清晰、描述性的语言
   - 用空行分隔多个提示词

2. **批量处理**
   - 从较少的并发提示词开始测试
   - 根据您的需求调整延迟
   - 监控进度部分以了解状态

3. **图片管理**
   - 清晰命名图片以便自动匹配
   - 使用支持的格式（PNG、JPG、GIF）
   - 保持文件大小在 10MB 以下

4. **性能**
   - 较少的并发提示词 = 更稳定
   - 更高的延迟 = 更少的服务器负载
   - 为下载使用适当的质量设置

## 🔧 故障排除

| 问题                     | 原因和解决方案                                                                                                                                                               |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **扩展程序不工作**       | 确保您在 Google Flow 项目页面上。检查扩展程序是否已启用。刷新页面并重试。                                                                                                 |
| **"队列已满"错误**       | Google Flow 限制同时处理的视频数量。**不用担心**，扩展程序将自动等待并每 30 秒重试一次，直到有可用位置。                                                                   |
| **视频不下载**           | 转到 Chrome 设置 -> 下载 -> **关闭** _"下载前询问每个文件的保存位置"_。检查设置标签页中的下载设置。                                                                         |
| **"政策错误"**           | 您的提示词或图片违反了 Google 的内容政策。工具将自动跳过此任务并继续下一个。                                                                                               |
| **生成失败**             | 检查您的互联网连接。验证提示词是否有效。查看重试设置。检查浏览器控制台以查找错误。                                                                                           |
| **屏幕缩小**             | 这是工具自动功能，用于覆盖界面并准确定位按钮。工具运行时请不要手动调整缩放。                                                                                                 |
